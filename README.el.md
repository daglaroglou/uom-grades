# 🎓 UoM Grades

*Παρακολούθηση βαθμών για το Πανεπιστήμιο Μακεδονίας*

Εφαρμογή desktop για προβολή και παρακολούθηση των βαθμών σας έξω από το πανεπιστημιακό portal.  
Συνδεθείτε μία φορά, διατηρήστε την πρόοδό σας με μια ματιά.

## ✨ Χαρακτηριστικά

### Πίνακας ελέγχου

| Χαρακτηριστικό | Περιγραφή |
|----------------|-----------|
| **Κάρτες σύνοψης** | Μέσος όρος, πρόοδος ECTS (→240), επιτυχημένα μαθήματα, καλύτερος βαθμός |
| **Βαθμοί ανά εξάμηνο** | Μαθήματα ομαδοποιημένα με σαφή ενδείξεις επιτυχίας/αποτυχίας |
| **Ιστορικό προσπαθειών** | Αναπτύξτε κάθε μάθημα για προηγούμενες εξεταστικές |
| **Κατανομή βαθμών** | Στατιστικά ανά μάθημα (μέσος, διάμεσος, ποσοστό επιτυχίας) με γραφήματα |

### Εμπειρία

| Χαρακτηριστικό | Περιγραφή |
|----------------|-----------|
| **Διατήρηση συνεδρίας** | Αποκαθιστά τη συνεδρία σας κατά την εκκίνηση |
| **Σκοτεινή λειτουργία** | Ενσωματωμένη εναλλαγή θέματος |
| **Δίγλωσσο** | English · Ελληνικά |
| **System tray** | Διατήρηση στο tray όταν κλείνει |
